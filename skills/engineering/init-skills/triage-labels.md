# Triage Labels

Maps canonical triage roles to the actual label strings used in this project's issue tracker. Edit the right-hand column to match your local vocabulary.

| Canonical role | Label string |
|---|---|
| needs-triage | needs-triage |
| needs-info | needs-info |
| ready-for-agent | ready-for-agent |
| ready-for-human | ready-for-human |
| wontfix | wontfix |

## Usage

When a skill refers to a canonical role (e.g. `needs-triage`), look up the label string from the right column before issuing any issue tracker command.

Edit the right-hand column to match what your project actually uses. For example, if your project uses `status: awaiting-feedback` instead of `needs-info`, change that row.
